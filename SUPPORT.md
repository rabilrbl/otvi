# Support

Use the right channel so questions and bugs are easier to triage.

## Where to Go

- Bug report: open a GitHub issue with the bug template
- Feature request: open a GitHub issue with the feature template
- Documentation problem: open a GitHub issue with the docs template
- Security concern: follow `SECURITY.md`

## What to Include

- what you were trying to do
- what you expected to happen
- what actually happened
- relevant logs, screenshots, or reproduction details
- version or commit, if known

## Before Opening an Issue

- check the README and docs site
- search existing issues first
- reduce the problem to a minimal reproduction when possible
